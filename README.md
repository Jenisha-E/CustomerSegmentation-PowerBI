# Power BI Customer Segmentation

## Project Overview
This project analyzes customer purchase data to identify segments based on age, spending patterns, and product preferences. Interactive dashboards in Power BI provide actionable insights for targeted marketing and business strategies.

## Dataset
Columns included:  
- CustomerName  
- Age  
- Product Category  
- OrderAmount  
- Date  

## Features
- Segment customers by age and spending  
- Identify top products and sales trends  
- Interactive dashboards for easy insights  
- Demonstrates Power BI proficiency  

## Tools Used
- Power BI Desktop  
- Excel for dataset  

## Steps Implemented
1. Data cleaning and formatting  
2. Exploratory Data Analysis (EDA)  
3. Customer segmentation using clustering  
4. Dashboard creation with charts and visuals  
5. Insights and recommendations  

## Outcome
- Clear understanding of customer groups  
- Insights for targeted marketing and product strategy  
- Enhanced business decision-making
